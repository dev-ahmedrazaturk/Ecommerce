FAQ

How do I integrate this with Django?

Place this file in templates/ and render it via a view. For example, map path('', TemplateView.as_view(template_name='landing.html')) in urls.py.

Can I use my own colors?

Yes—edit the CSS variables at the top (--primary, --accent, etc.).
